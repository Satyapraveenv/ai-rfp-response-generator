## 1. Company Background

HealthFirst Insurance Group is a mid-sized health insurance provider serving 2.8 million members across 12 US states. We operate a hybrid IT environment with legacy mainframe systems, modern cloud workloads on AWS, and 40+ microservices supporting our member portal, claims processing, and provider network.

We are seeking a **Quality Engineering and Digital Transformation partner** to modernize our testing practices, reduce release cycle time, and improve software quality across our portfolio.

---

## 2. Scope of Work

### 2.1 Quality Engineering Transformation
- Assess current QA maturity across 6 business-critical applications
- Design and implement a **Test Automation Framework** (web, mobile, API, and performance)
- Migrate from manual-heavy testing (70% manual, 30% automated) to **80%+ automation coverage**
- Establish shift-left testing practices integrated into CI/CD pipelines
- Implement AI-assisted test generation and defect prediction

### 2.2 DevOps & CI/CD Enablement
- Set up automated pipelines for 3 flagship products (member portal, claims engine, provider directory)
- Integrate automated testing into Jenkins/GitHub Actions pipelines
- Implement code quality gates (SonarQube, SAST/DAST scanning)
- Target: **release cycle reduction from 6 weeks to 2 weeks**

### 2.3 Performance & Security Testing
- Establish a performance testing center of excellence
- Load, stress, and soak testing for peak enrollment periods (open enrollment: Oct-Dec)
- HIPAA compliance testing and security vulnerability scanning
- Target: support **50,000 concurrent users** during open enrollment

### 2.4 Cloud Migration Testing Support
- Test planning and execution support for AWS migration of 3 legacy applications
- Data migration validation and reconciliation testing
- Rollback testing and disaster recovery validation
